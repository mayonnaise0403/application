李昱醇的申請表

個人簡介

        我是李昱醇，來自輔仁大學日文系的大四生。我一直都對未來沒有想法，只知道盲目地讀要考的內容。在大學這三年，身邊許多人去了實習、去了日本交換留學，而沒有輔系和雙主修的我還在原地不知道未來要做甚麼。在我感到迷茫的時候，家人建議我可以去學程式設計，如果有問題可以去問我姐或我爸，他們也都是工程師，結果學了一陣子，發現自己對程式還蠻有興趣的，就開始不斷地利用網路資源自學，喜歡自己解決問題時的成就感，因此就開始朝著工程師之路前進。

為什麼想成為前端、後端、或全端工程師？

        因為家人的建議，我開始學習程式，使得我的生活產生了巨大的改變，變成了大部分時間在學習程式，讓漫無目的的我，開始有了明確的目標，就是要成為後端的工程師。

        我後來就開始在網路上爬文自學，從HTML、CSS、RWD、JavaScript開始入手，雖然目標是後端，但是聽說前端比較好入門，所以先從前端開始學習，我本以為我會很快就放棄，沒想到自己還學得挺開心的，才發現這才是真正我要的、我喜歡的東西，在學習過程中雖然也遇到了不少的挑戰，但自己並不排斥解決問題，希望自己能夠接受更艱難的挑戰，所以才會選擇此訓練營，希望能為自己打造一個高壓的環境，讓自己有個突破的機會，希望能藉由自己的努力，讓我在畢業後能夠順利轉職，因為在老師這裡畢業後是明年三月中，差不多大學也快畢業了，就可以開始找正職或是實習，希望能夠在畢業後無縫接軌，我也很期待在第二階段可以體驗一下和同學一起合作開發小專案。

為了成為軟體工程師，曾經做過什麼努力？

        為了成為工程師，我花了很多的時間在自學上面，一開始我是看網路上的教學影片來自學的，剛好自己也不排斥英文，所以我就找到了W3schools的英文課程來自學，遇到不懂就是不斷的google，直到問題解決為止，每次自己成功的解決問題，都會讓我更有動力的持續學習下去。

        在自學的方面，大概有了 HTML 、 CSS 、 JavaScript 的基礎，在看到訓練營的第一階段需要用到 python 時，我就開始學習 python ，利用 python 的 tkinter 和 pygame 的模組做了一些小工具和遊戲，像是 BMI 計算機還有簡易的雙人乒乓球對打遊戲，目前正在利用 pygame 模組，著手開發一個小型的 RPG 遊戲，下面的影片是目前的製作進度，可以控制打開背包，還有跟樹怪對話，還有撿道具，目標是製作一個遊戲時數一小時以上的遊戲，不過我有詢問之前的學員 Juri ，他說在訓練初期是不允許使用任何套件的，所以等我完成這份作品，我想要不使用套件再來寫寫看，使用原生寫法來解決問題。

        除了免費的課程，我還有去上彭彭老師在台大的364期的JavaScript(ES6)網頁前端工程入門，解決了許多我以前以為自己懂，結果上了老師的課才發現「哦~~原來是這樣~」的地方還蠻多的，真的教的非常仔細，讓我受益良多。

        雖然到目前為止大部分的時間都是靠著自學來累積實力，目前也有計畫開發小遊戲，但是總覺得還是缺少了什麼，缺少了高壓的環境，通常在高壓還有時間的壓力下，會激發人的潛能，所以我希望能夠藉由WeHelp Bootcamp的訓練營，來幫助我來打造我想要的環境，我相信在這樣的環境下，肯定能夠激發出不一樣的自己。

如果參與這個計畫，會怎麼安排學習時間？

        我自己本身平常就是早上花兩小時左右學習本科的日文，畢竟外語也是自己的一項技能，我也不打算放棄它，而其餘時間就會開始學習寫程式。平常一天都會花七、八小時在學習程式上面，每次在思考一些小專案的時候，常常一抬頭就發現一小時一下就過了，大部分的時間是在思考和解決問題。因為要升大四的關係，所以在學習的時間上面我是有非常多的時間可以利用的。自己也已經長時間養成這樣的學習習慣了，在立志要當工程師後，得到了許多家人與朋友的鼓勵，就開始每天不斷的學習，也希望自己能有一天能夠成功當上工程師。

         雖然目前我才剛要升上大四，但是我大四只有禮拜四有課程，而且總共只有四小時左右，其實也不太會影響到我學習程式的時間。雖然說有畢業專題，但我早就在七月暑假期間還有大三的時候就完成了，所以時間可以說是十分充裕，因為想要好好的利用大四的這段空閒時間，所以才會選擇報名今年的訓練營，並且希望能夠訓練自己越來越早交出作業，訓練自己越來越快完成任務，我相信我自己絕對可以全心投入課程， 我已經準備好接受老師的魔鬼訓練了!!

如果參與這個計畫，預期會碰到什麼困難？你打算怎麼解決它？

程式一直出錯 : 基礎學完後，做一些小作品的時候，常常會跑不出來，或是沒有自己想要的結果。

解決方法:現在科技那麼進步，我覺得最好的解決辦法就是去google找答案，到目前為止我所遇到的困難都是可以靠google解決的，所以我相信用google找答案會是個最好的解決辦法，在訓練營的課程裡面，一定會有我沒有學過的東西，所以可能要花更多的時間去google還有思考，不過到目前為止的所有問題，都是有成功解決的，只是有些花了比較久的時間，甚至還有想了兩天左右才解決的，連吃飯都在想要怎麼解決，雖然解決的過程很辛苦，但是一旦解決，就會覺得一切都值得了。


壓力大如何排解:任務太難，沒辦法在短時間內完成的時候。

解決辦法:我覺得壓力大的時候，要讓自己腦袋清淨一下，像是我有時候花了好久，才終於解決問題的時候，我就會讓自己去休息，讓自己腦袋休息一下，不要再寫了，通常自己解決一個大問題的時候，就會去玩玩遊戲，放鬆自己，不要讓自己腦袋爆炸，如果想不出來的時候，我會把當天想的邏輯寫下來，當天先暫時不寫這部分的邏輯，先寫其他的，明天再來思考，有時候休息一下，就會突然想出來。

是否有想要加入的軟體公司？為什麼想加入該公司？

        其實我自己本身很喜歡玩遊戲，所以我希望未來可以進去遊戲公司，做遊戲的開發，我目前的目標是希望能夠進 Garena 的公司做開發，雖然可能不是那麼的容易，但是我目前以這個為目標，等到我有一定能力的時候，我想要自己開發遊戲，因為常常玩遊戲都會遇到我覺得不方便的功能，所以我的夢想是希望能夠找到幾個同仁，開個小工作室，開發遊戲，做我喜歡的遊戲。

想要對我們說的事情？

        我很想跟老師說，謝謝您願意花很多時間在教育上面，對於沒有相關背景的我們，想要學習程式可能會有點困難，但是老師給了我們非常多新手友善的教學資源，像是台大的資訊系統訓練班的課程，還有YouTube頻道等等。許多很基礎的地方都是從老師的頻道學習的，淺顯易懂的內容真的對我幫助很大，謝謝老師!!
